Sobre este Proyecto

Este proyecto fue desarrollado como una guía de clase para implementar un formulario de Login funcional usando una Petición POST con Fetch, con validaciones básicas en el servidor (Backend).

La arquitectura separa el Frontend y el Backend utilizando la tecnología CORS para la comunicación.
